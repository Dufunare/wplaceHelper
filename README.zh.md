# wplaceHelper
一个基于 Python 的 [wplace.live](https://wplace.live) 辅助工具 💖

## ✨ 功能特色
- 🖼️ 本地图片像素化并即时预览
- 🎨 右键点击每个像素，可标记该像素在网页上是否已绘制
- ⚙️ 支持多种算法和尺寸进行图片像素化
- 💾 支持项目保存，方便后续继续创作
- 📤 支持将像素化后的图片导出到本地目录
- 🚧 更多功能即将推出...

## 🚀 安装方法
**需要 Python 版本：3.10+**  
**推荐在虚拟环境中运行**  
步骤：  
1. 克隆本项目到本地目录
2. 运行 run.bat
